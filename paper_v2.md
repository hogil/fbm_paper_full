# Failbit Map 불량 패턴 분류를 위한 지도-비지도 복합 파이프라인 설계

**A Hybrid Supervised-Unsupervised Pipeline for Semiconductor Wafer Failbit Map Defect Pattern Classification**

---

## 초록

반도체 제조 공정에서 웨이퍼 Failbit Map의 불량 패턴 분류는 수율 관리의 핵심 과제이나, 기존 CNN 기반 방법은 라벨 오류 누적과 미등록 신규 불량 대응에 한계가 있다. 본 연구에서는 등록 불량에 ConvNeXtV2 + Grad-CAM + YOLO 2단계 구조를 적용하여 weighted F1 0.95를 달성하고, 미등록 불량에는 자기지도 대조 학습과 HDBSCAN 군집화를 결합한 개방집합 탐지 모듈을 구현하였다. 두 파이프라인은 전문가 검토 기반 라벨 확장 루프로 연결되며, 칩별 전기적 특성 검사 결과와 웨이퍼 수율 정보를 이미지와 정합 저장하여 향후 멀티모달 분석으로의 확장 기반을 마련하였다.

---

## 1. 서론

반도체 메모리 제조 공정에서 Failbit Map은 웨이퍼 내 각 셀의 전기적 테스트 결과를 공간적으로 표현한 정보로, 불량 패턴의 위치와 분포 형상은 공정 이상 원인 추적에 핵심적인 단서를 제공한다 [1]. center, donut, edge ring, scratch 등 반복적으로 관찰되는 패턴은 오랫동안 규칙 기반 또는 CNN 기반 지도학습으로 분류되어 왔으며 [2], 웨이퍼 단위의 자동 분류는 수율 관리의 중요한 요소로 자리잡고 있다.

본 연구는 처음에 기등록 불량을 CNN으로 분류하는 문제로 시작하였다. 그러나 실제 현장 적용 과정에서 단일 지도학습 모델만으로는 해결되지 않는 세 가지 현실적 한계를 확인하였다. 첫째, 전문 엔지니어의 판독에 의존하는 라벨링 과정에서 오류가 학습 데이터에 누적되며, 이는 모델 성능을 잠재적으로 저하시킨다. 둘째, 신규 제품 도입이나 공정 조건 변화로 인해 기존 분류 체계에 없는 불량 패턴이 발생하더라도, 폐쇄집합(closed-set) 분류기는 이를 강제로 기존 클래스에 배정하여 오진을 야기한다. 셋째, Failbit Map 단독 이미지 분석만으로는 설명하기 어려운 전기적 특성 정보와의 결합 분석에 대한 현업 요구가 존재한다.

이에 본 연구에서는 이러한 문제를 도메인 지식 기반의 창의적인 설계로 해결하는 통합 파이프라인을 제안한다. 구체적인 기여는 다음과 같다.

- **2단계 등록 불량 분류**: 웨이퍼 불량의 공간적 위치 특성을 반영한 Grad-CAM ROI 추출과 클래스별 공간 사전 정보(spatial prior), YOLO 이차 검증을 결합하여 라벨 오류 자동 탐지 및 분류 성능 향상
- **미등록 불량 자동 발굴**: 동결 백본 재사용, Global/Local InfoNCE, HDBSCAN 밀도 군집화를 결합한 자기지도 군집화 파이프라인 설계
- **멀티모달 데이터 인프라 구축**: 칩별 전기적 특성 검사 결과 및 웨이퍼 수율 정보를 이미지와 정합 저장하는 데이터 파이프라인을 통해 향후 이미지-계측 융합 분석의 기반 마련

### 2.1 2단계 지도학습 파이프라인 (등록 불량 분류)

**1단계 — ConvNeXtV2 분류 및 Grad-CAM ROI 추출**

ConvNeXtV2-Base [3]를 백본으로 사용하여 384×384 해상도의 웨이퍼 레벨 Failbit Map으로부터 16개 클래스(center, donut, edge_loc, edge_ring, local, random, scratch, near_full, none 등)를 분류한다. 분류와 동시에 Grad-CAM [4]을 적용하여 예측 클래스에 대한 공간 활성화 맵을 생성하고, 이를 형태학적으로 정제하여 불량 위치의 ROI(Region of Interest)를 추출한다.

**신뢰도 임계값(Confidence Threshold) 설정**

CNN 예측 신뢰도를 기준으로 ROI 보강 적용 여부를 결정하는 `CONFIDENCE_THRESHOLD`는 0.5~0.95 구간을 다양하게 실험하여 최종 **0.80**으로 설정하였다. 임계값이 낮을수록 많은 샘플에 ROI 보강이 적용되어 연산 비용이 증가하고, 이미 신뢰도 높게 맞은 샘플에 불필요한 보정이 가해져 오히려 성능이 저하되는 경향이 관찰되었다. 반면 임계값이 높을수록(≥ 0.90) 난이도 높은 샘플에 보강이 적용되지 않아 개선 효과가 감소하였다. 클래스 단위 판단은 별도 `PRECISION_THRESHOLD`(= 0.80)를 기준으로 precision/recall 이 모두 해당 기준 미만인 클래스를 **difficult class**로 자동 지정하고, 해당 클래스 예측 샘플에는 신뢰도와 무관하게 ROI 보강을 적용한다. 두 조건(low-confidence OR difficult class) 중 하나라도 해당하면 보강을 수행하는 선택적 설계로 연산 효율을 확보하였다.

여기서 반도체 도메인 지식이 핵심적으로 활용된다. 웨이퍼 불량은 패턴 종류에 따라 발생 위치가 다르다는 특성—center는 웨이퍼 중심부, edge_ring은 외곽 테두리, scratch는 특정 방향의 선형 구조—을 반영하여, 학습 데이터 전체에서 클래스별 평균 ROI 좌표를 사전 학습한다. 이 공간 사전 정보는 모델 신뢰도가 낮거나 difficult class에서 동적 Grad-CAM ROI와 혼합(α 블렌딩)하여 ROI 추출의 안정성을 높인다.

**2단계 — YOLO 이차 검증 및 라벨 오류 교정**

추출된 ROI를 YOLO [5]에 입력하여 이차 검증을 수행한다. 1단계 분류 결과와 YOLO 검출 결과가 불일치하는 샘플은 라벨 오류 후보로 분류된다. 이 과정은 단순한 분류 보강을 넘어 기존 학습 데이터의 라벨 오류를 시스템적으로 탐지하는 역할을 수행하였다. 실제로 이 2단계 구조를 통해 weighted F1 점수가 0.92에서 0.95로 향상되었으며, 향상의 핵심 원인이 라벨 오류 탐지 및 교정에 있음을 확인하였다.

YOLO 학습 및 추론 단계에서도 다양한 하이퍼파라미터를 조정하였다. **추론 신뢰도 임계값**(`YOLO_CONF_THRESHOLD`)은 0.1~0.7 구간을 실험하였으며, 0.25 미만에서는 배경 영역의 오탐이 증가하고 0.5 이상에서는 소형·저밀도 불량 패턴의 탐지가 누락되어 최종 **0.25**로 설정하였다. 학습 단계에서는 이미지 크기(imgsz), 배치 크기, 학습률, 데이터 증강 조합(mosaic, HSV 변환 강도, rotation)을 조정하였다. 특히 CNN과 동일한 도메인 원칙을 적용하여 **좌우·상하 반전 증강을 비활성화**하였다—반전은 edge_loc의 발생 방향, scratch의 진행 방향 등 공간 방향 정보를 파괴하여 탐지 성능을 오히려 저하시키는 것을 실험으로 확인하였다. 이 반전 제거가 YOLO 단독 탐지 mAP 기준으로 약 3%p 향상에 기여하였다.

### 2.2 자기지도 대조 학습 + HDBSCAN (미등록 불량 탐지)

**모델 구조 — 동결 백본 전이**

미등록 불량 탐지 모듈은 등록 불량 분류에서 학습된 ConvNeXtV2 백본을 동결(freeze)하여 특징 추출기로 재사용하고, 그 위에 2층 MLP 투영 헤드(128-dim, L2 정규화)만 새로 학습한다. 이를 통해 기등록 불량 이미지로부터 이미 학습된 웨이퍼 맵 공간 표현을 보존하면서, 최소한의 추가 학습으로 미등록 불량에 대한 임베딩 공간을 구성할 수 있다.

**손실 함수 — Global InfoNCE + Queue + Local InfoNCE**

대조 학습 손실은 두 가지로 구성된다. Global InfoNCE [6]는 16,384개 크기의 메모리 큐를 활용하여 대규모 음성 샘플을 효율적으로 사용하며, 코사인 유사도 임계값(0.72) 이상인 쌍은 음성 집합에서 제외하여 외형이 유사한 서로 다른 불량 패턴이 과도하게 반발되는 위음성(false negative) 문제를 방지한다.

Local InfoNCE는 피처 맵 위에 6×6 격자 앵커를 설정하고, 윈도우 내 국소 위치 간 유사도를 학습한다. Failbit Map에서 불량은 특정 공간 영역에 집중 발생하는 특성이 있으므로, 전역 표현만으로는 포착하기 어려운 국소 결함 구조를 지역 대조 손실로 보완하는 것이 설계 의도이다.

**도메인 특화 증강**

일반적인 이미지 학습에서 흔히 쓰이는 좌우·상하 반전을 적용하지 않는다. 반전은 웨이퍼 불량의 절대적 공간 위치 관계를 변형하여—예컨대 edge_loc 패턴의 발생 방향, scratch의 진행 방향—불량 유형을 구분하는 표현 학습을 방해하기 때문이다. 대신 소폭의 회전(±7°), 이동(±5%), 스케일 변화(±5%), 가우시안 노이즈만을 적용하여 공간 방향 정보를 보존하면서 촬영 조건 변화에 대한 불변성을 학습한다.

**HDBSCAN 군집화 및 품질 필터링**

학습된 임베딩 공간에서 HDBSCAN [7]을 적용하여 미등록 불량을 자동 군집화한다. K-means와 달리 클러스터 수를 사전에 지정할 필요가 없고 잡음 포인트를 명시적으로 처리하여, 발생 빈도가 제각각인 미등록 불량 환경에 적합하다. 클러스터 품질 필터링은 최소 크기(12개), 연성 멤버십 확률 중앙값(≥ 0.55), 계층적 지속성(≥ 0.20) 기준을 함께 적용하며, 각 군집에 대해 medoid 대표 이미지와 군집 요약 정보를 자동 저장하여 전문가의 신속한 검토를 지원한다.

**Fig. 2. Failbit Map 데이터 생성 파이프라인**

Bucket A·B 파일명 패턴 매칭과 ±10초 시간 윈도우 기반 동기화를 통해 두 버킷의 데이터를 칩 단위로 정합한다. 대용량 병렬 처리(최대 256 연결)와 Cython으로 컴파일된 Hex 파싱 모듈로 실시간에 가까운 맵 생성 속도를 달성하였다.

**32색 팔레트 인덱스 PNG — 화질·용량·커스터마이징의 동시 달성**

Failbit Map 이미지는 JPEG, 일반 24bit PNG 대신 8비트 팔레트 인덱스 PNG(color type 3)를 채택하였다. JPEG는 DCT 기반 손실 압축으로 Grade 경계가 뭉개져 Grade 0~7 구분 자체가 불가하며, 24bit PNG는 화질은 유지하지만 사용자별 색상 변경 시 수십만 장 전체를 재렌더링해야 하는 운영 부담이 발생한다. 팔레트 인덱스 PNG의 IDAT 청크에는 색상이 아닌 의미적 인덱스(Grade·BIN 번호)만 저장되고, 색상 정의는 PLTE 청크(96 bytes, 32색)에만 존재하므로, **PLTE만 교체하면 IDAT 재압축 없이 즉각 색상 전환**이 가능하다.

이 설계는 반도체 현장 운영 관행에서 도출된 것이다. 현업 엔지니어들은 불량 BIN별로 개인화된 색상 체계를 수십 년간 사용해왔으며, 같은 웨이퍼 맵이라도 담당자마다 서로 다른 색상 배정을 요구한다. 팔레트 인덱스 구조를 통해 동일 원본 이미지 파일 하나로 사용자별 맞춤 색상을 실시간으로 서빙할 수 있으며, 24bit RGB 대비 약 65% 파일 크기 절감과 화질 완전 보존을 동시에 달성하였다. 이는 도메인 지식을 파일 포맷 설계에 직접 적용한 사례이다.

### 2.4 Human-in-the-loop 라벨 확장

HDBSCAN 군집화 결과는 자동 의사결정의 종점이 아닌, 전문가 검토를 위한 후보 생성 단계로 활용된다. 담당자는 군집 대표 이미지, 군집 요약 통계, Failbit Map 시각화, 칩별 계측값 오버레이를 함께 확인하면서 신규 불량 여부를 판단한다. 확정된 군집은 새로운 클래스로 등록되어 지도학습 파이프라인의 학습 데이터에 즉시 편입되며, 이를 통해 시스템이 공정 변화에 지속적으로 적응하는 피드백 루프가 형성된다.

## 3. 실험 결과

실험은 사내 Failbit Map 웨이퍼 레벨 이미지 데이터셋을 대상으로 수행하였다.

### 3.1 등록 불량 분류 — 성능 향상 과정

**Table 1. 모델 구성별 성능 향상 단계 (Weighted F1)**

| 단계 | 구성 | Weighted F1 | 주요 변경 |
|:---:|------|:-----------:|---------|
| 1 | 경량 CNN (기준선) | ~0.78 | ResNet 계열 소형 모델, 기본 하이퍼파라미터 |
| 2 | ConvNeXtV2-Base | ~0.85 | 고용량 백본 교체, Optuna 1차 탐색 |
| 3 | + 입력 해상도 384×384 | ~0.88 | 모델 사전학습 해상도에 맞춤 |
| 4 | + FCMAE·ImageNet-22K 사전학습 | ~0.90 | 대용량 비지도 사전학습 가중치 적용 |
| 5 | + Optuna 집중 튜닝 + 데이터 정제 | **0.92** | LR·WD·scheduler·augment 최적화, 라벨 정제 |
| 6 | + YOLO 이차 검증 + ROI 보강 | **0.95** | 라벨 오류 교정 + 2단계 구조 |

단계별 핵심 포인트는 다음과 같다.

- **백본 선택(2단계)**: EfficientNet, ResNet 등과 비교 실험 결과 ConvNeXtV2-Base가 웨이퍼 맵 공간 패턴 학습에서 우위를 보였다. 대용량 커널과 깊은 계층 구조가 center, edge_ring처럼 넓은 공간 범위 패턴 표현에 유리하기 때문으로 해석된다.
- **고해상도 입력(3단계)**: 384×384 입력은 scratch·local 패턴의 미세 구조 구분에 결정적이었으며, 224×224 대비 fine-grained 특징 보존 효과가 명확하였다.
- **Optuna 집중 튜닝(5단계)**: Optuna TPE Sampler를 사용하여 학습률(LR), 가중치 감쇠(weight decay), 학습률 스케줄러(cosine annealing vs step), 드롭아웃 비율, 레이블 스무딩 계수, 데이터 증강 강도(RandAugment magnitude, cutmix alpha) 등 복합 하이퍼파라미터 공간을 체계적으로 탐색하였다. 단일 파라미터 수동 탐색 대비 실험 효율이 크게 향상되었으며, 이 단계에서 0.90 → 0.92로의 도약이 이루어졌다. 또한 클래스별 F1 모니터링을 통해 difficult class를 식별하고 해당 데이터를 재검토·보강하였다.
- **2단계 구조(6단계)**: Grad-CAM ROI 추출, 클래스별 공간 사전 정보(spatial prior) 학습, α-블렌딩, 선택적 ROI 적용(confidence < 0.80 또는 difficult class 한정), YOLO 이차 검증(conf ≥ 0.25, 반전 증강 제거)을 결합하였다. 16개 클래스 전체에 걸쳐 일관된 임계값 적용으로 difficult class 자동 식별 및 교정이 가능하였다.

### 3.2 미등록 불량 군집화 — 튜닝 기법 및 평가

**구현 기법과 효과**

대조 학습 + HDBSCAN 파이프라인에서 미등록 불량 군집 품질을 높이기 위해 적용한 주요 설계 선택과 그 근거는 다음과 같다.

| 기법 | 설정값 | 효과 |
|------|--------|------|
| 동결 백본 전이 | FREEZE_BACKBONE=True | 소규모 unknown 데이터에서도 안정적 임베딩 |
| Memory Queue | QUEUE_SIZE=16,384 | 배치 크기 한계를 넘는 대규모 음성 비교 |
| False negative 마스킹 | IGNORE_NEG_SIM=0.72 | 유사 불량 간 과반발 억제 |
| Local InfoNCE | grid36_full, window=4, topk=12 | 국소 결함 위치·형태 정보 보존 |
| NO flip 증강 | 좌우·상하 반전 제외 | 공간 방향 정보(불량 위치) 보존 |
| 클래스 균형 샘플링 | ratio=0.25, min=1 | 다수 클래스 편향 억제 |
| HDBSCAN leaf 선택 | epsilon=0.06 | 과분할 방지, 안정적 군집 경계 |
| 군집 품질 필터 | size≥12, prob≥0.55, persist≥0.20 | 잡음 군집 자동 제거 |

**미등록 불량 성능 평가 방법론**

미등록 불량은 정의상 사전 Ground Truth를 구성하기 어려워, 지도학습과 동일한 방식의 정량 평가가 불가능하다. 이 문제에 대응하는 평가 접근법은 크게 세 층위로 구성할 수 있다.

*(1) 레이블 없는 내부 지표:* Silhouette Score는 군집 내 결집도와 군집 간 분리도를 동시에 측정하며, 값이 높을수록 임베딩 공간에서 패턴이 명확히 구분됨을 의미한다. HDBSCAN의 연성 멤버십 확률(soft membership probability) 분포도 군집 응집도의 대리 지표로 활용 가능하다.

*(2) 전문가 동의율(Human Agreement Rate):* 각 군집을 담당 엔지니어가 검토하여 유효한 신규 불량 유형으로 확정한 비율을 측정한다. 이는 군집화 결과의 현업 유용성을 직접 반영하는 지표이다.

*(3) 하위 태스크 전이 성능:* 전문가가 군집에 라벨을 부여한 후 해당 클래스를 등록 불량 파이프라인에 추가하여 재학습하였을 때의 F1 향상을 측정한다. 군집화 품질이 높을수록 라벨 확장 후 F1 향상 폭이 크게 나타난다.

**Table 4. HDBSCAN 주요 파라미터 구성 비교**

| 구성 | min_cluster_size | epsilon | 결과 군집 수 | 무시 샘플 비율 |
|------|:---------------:|:-------:|:---------:|:-----------:|
| 세분화(aggressive) | 5 | 0.02 | 多 | 낮음 |
| 기본(채택) | **12** | **0.06** | **적정** | **적정** |
| 보수적(conservative) | 25 | 0.15 | 少 | 높음 |

채택된 기본 구성은 군집 수와 무시 샘플 비율 간의 균형을 최적화한 결과이며, 현업 검토 부담과 탐지 커버리지를 동시에 고려하였다.

## 4. 결론

본 연구에서는 반도체 Failbit Map의 웨이퍼 레벨 불량 패턴 분석을 위해 지도-비지도 복합 파이프라인을 제안하였다. ConvNeXtV2 + Grad-CAM + YOLO 기반 2단계 구조로 등록 불량 분류에서 weighted F1 0.95를 달성하였으며, 자기지도 대조 학습과 HDBSCAN을 결합하여 미등록 불량 패턴의 자동 군집화 기능을 구현하였다. 또한 팔레트 인덱스 PNG 기반 고속 시각화, 칩별 전기적 특성 검사 결과 및 웨이퍼 수율 정보의 정합 저장을 통해 멀티모달 분석으로의 확장 기반을 마련하였다.

후속 연구는 세 방향으로 계획하고 있다. 첫째, 현재 웨이퍼 레벨 분류를 칩 레벨 불량 국소 분석으로 확장하고 다중 불량 패턴이 공존하는 상황을 처리하는 다중 라벨(multi-label) 분류 체계를 도입한다. 둘째, 본 연구에서 구축한 칩별 전기적 특성 검사 결과와 이미지 임베딩을 결합하는 멀티모달 불량 분석 모델을 설계한다. 셋째, 미등록 불량 군집화에 대한 정량적 평가 프로토콜을 수립하고 더 넓은 제품군과 공정 라인으로의 일반화 가능성을 검증한다.
