# Word Submission Rules

## Page Setup

- Paper size: `A4`
- Top margin: `3.0 cm`
- Bottom margin: `2.5 cm`
- Left margin: `1.5 cm`
- Right margin: `1.5 cm`
- Gutter: `0 cm`

## Fonts And Line Spacing

- English font: `Times New Roman`
- Korean font: `바탕체`
- Line spacing: `1.0`

## Font Sizes

- Title: `20 pt`, `bold`
- Author names: `12 pt`, `bold`
- Abstract heading: `10 pt`, `bold`
- Section headings: `11 pt`, `bold`
- Body text: `10 pt`
- Figure and table captions: `9 pt`, `bold`
- References: `9 pt`

## Layout

- Columns: `2`
- Column spacing: `0.5 cm`
- Title and abstract must be placed outside the 2-column body area, using single-column full width.
- Leave one blank line above and below section headings, tables, figures, and equations so they are visually separated from the body text.

## Author Formatting

- List the first author first, then co-authors in descending order of contribution.
- Put affiliation information separately at the bottom of the first page.
- Write affiliation in this order:
  `Department, Organization, City, Country`
- Use superscript numbers after author names and before affiliations, and match the same number to the same affiliation.

## Tables And Figures

- Table titles go above the table.
- Figure titles go below the figure.
- Align both table and figure titles to the left.

## Citation And References

- In the body text, cite references with square brackets on the right side of the cited sentence, for example: `[1]`
- Assign reference numbers in the order they first appear in the body text.

## Length

- Abstract: up to `2 pages`
- Full paper: `4-10 pages`

## Notes

- This rule sheet was normalized from user-provided instructions and will be used as the default formatting guide for future `.docx` generation in this project.
