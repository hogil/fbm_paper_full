# 부서장 추천서

## 1. 보유역량

ㅁ **기술 전문성** (AI 알고리즘에 대한 깊은 이해를 바탕으로, 현장의 데이터를 분석하여 최적의 모델을 직접 설계하고 구현 (coding) 할 수 있는 실무 능력)

현장에 AI 를 적용 / 운영하기 위해서는 데이터뿐만 아니라 업무 프로세스, 비즈니스 가치에 대한 전반적인 이해가 필요함. 최호길님은 10년의 현장 경험을 갖추어 이러한 이해도가 매우 우수함. AI 알고리즘에 대한 깊은 이해를 갖추기 위해 2024년 서울대학교에서 Data Scientist 인증을 취득하였고, 현재 연세대학교에서 AI 석사 과정을 업무와 병행하고 있음. 문제 정의를 스스로 할 수 있으며, 데이터의 특성에 맞춰 최적의 모델을 직접 설계 / 구현 / 검증까지 수행하여, 실제 시스템을 사용하는 현업 엔지니어로부터 인정을 받는 성과를 창출함.

ㅁ **기술 리더십** (본인의 AI 지식과 기술을 조직 내에 전파하고, 동료들의 기술적 난제를 해결하거나 팀의 AI 역량 상향 평준화를 주도하는 능력)

최호길님의 기술 리더십은 구체적인 기술 조언과 지식 전수에 강점이 있음. 동료들에게 model 선택, fine-tuning, 학습 안정화 전략을 조언하고, trend anomaly PoC 에서는 딥러닝 코드를 한 줄씩 직접 짚어 동료가 스스로 모델 학습과 데이터 생성까지 가능한 수준에 도달하게 함. 2024년 DRAM YE 팀장 주관 품질 시스템 회의, 2025년 MI 기술팀장 주관 D1c Ramp-up 회의 및 DRAM PA / Defect 제어팀 공청회에서 직접 발표 / 시연하였고, Confluence / GitHub / 사내 세미나로 최신 AI 기술 및 데이터 / GPU / 서버 활용 노하우를 상시 전파하고 있음.

ㅁ **기술 성과** (AI 기술을 실제 현장에 적용하여 수율 향상, 원가 절감, 시간 단축 등 비즈니스 가치를 창출한 실질적 결과물)

AI 기반 L3 Failbit Map 불량 분석 시스템 개발을 기획부터 프레임워크 개발, 데이터 / AI / 시스템 구축 및 운영까지 주도적으로 수행하였으며, DRAM YE 팀, EDS 팀, AI 센터와 협업하여 실제 현장에서 사용할 수 있도록 web app 을 open 함. DRAM D1a/b/c/d 분석 파트에서 매일 사용 중이며, 공수 **90% 이상 절감 (연 26억 효과)**, 수율 **+0.02% 개선 (P3WN 1건 97억 효과)** 의 성과를 창출함. 해당 결과물은 DS 부문 성과로 인정받아 **AI 센터 주관 DS AI Best Practice Good Challenger 상** 수상 및 **MTC 고등급 (1등급) 제안** 채택으로 이어짐.

## 2. 인증 후 기대 효과

(AI Specialist 인증 이후, 해당 인력이 부문 / 부서의 AI 전환 (AI Transformation) 을 가속화하는 데 기여할 미래 가치)

최호길님은 현재 메모리제조기술센터 직속 품질 부서인 QIE 그룹 Data Science 파트 소속으로, 10년의 현업 (Photo 기술팀, DRAM PA / YE) 경험에 기반하여 현장에 필요한 AI 시스템 구축을 주도적으로 수행하고 있음. AI Specialist 인증 이후에도 QIE 그룹의 중장기 과제인 제조 품질 AX (AI Transformation) 를 가속화하는 데 중추적인 역할을 하리라 기대함.
